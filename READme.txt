Muted Block Theme Package
=========================

FILES INCLUDED:
---------------
1. styles.css
   - Full, commented version with organized sections:
     • Theme Setup & Base Layout
     • Header, Footer & Global Elements
     • Cards, Grids & Tables
     • Components — KO Pages (A & G)
     • Animations & Responsive Layout

2. styles.min.css
   - Minified version for production use
   - Identical functionality, optimized for speed

3. README.txt
   - This file (usage guide)

USAGE:
------
For development:
<link rel="stylesheet" href="styles.css">

For production (faster load):
<link rel="stylesheet" href="styles.min.css">

NOTES:
------
• All pages (A-KO.html, G-KO.html, etc.) use the master stylesheet.
• No inline or extra CSS is needed.
• Theme automatically supports light/dark mode.
• All components (.ko-grid, .resource-section, .btn, .pdfModal) are built in.
