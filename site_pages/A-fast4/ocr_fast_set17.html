FAST FOUR STARTER

1. Explain

[1]

One benefit of automation in software testing.

Answer: • Allows large suites of tests to be run quickly and repeatedly with minimal manual effort
• Makes regression testing easier whenever code changes are made
• Improves consistency because the same tests are executed in the same way each time
• Frees testers to focus on exploratory and usability testing

2. State

[1]

One drawback of using virtual memory.

Answer: • Accessing pages from secondary storage (e.g. disk) is much slower than RAM, which can reduce performance
• Heavy use of paging can lead to thrashing, where the system spends most of its time swapping pages
• Requires extra hardware support (MMU) and operating system complexity
• Can make performance less predictable for time‑critical applications

3. Describe

[1]

One benefit of open standards.

Answer: • Promote interoperability, allowing products from different vendors to work together
• Reduce vendor lock‑in because users are not tied to proprietary formats or protocols
• Encourage competition and innovation by letting multiple companies implement the same standard
• Make it easier to integrate systems and share data across organisations

4. Explain

[1]

One benefit of virtual memory.

Answer: • Provides each process with a large, continuous logical address space regardless of physical RAM size
• Supports more concurrent processes by swapping inactive pages to secondary storage
• Increases system stability by isolating processes’ memory from each other
• Allows large applications to run on systems with limited physical memory
