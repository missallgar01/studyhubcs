1. Describe

One disadvantage of version control systems.

Answer: • Adds tools and workflows that developers must learn, increasing initial complexity
• Poorly managed branching and merging can create conflicts and slow development
• Storing many versions and branches consumes storage and may require server infrastructure
• Can give a false sense of security if teams rely on version history instead of proper testing and review

2. State

[1]

One benefit of defensive design.

Answer: • Anticipates incorrect or malicious input and handles it safely
• Includes validation, authentication and clear error messages to protect the system
• Reduces the likelihood of crashes and security vulnerabilities
• Improves reliability and robustness of software in real‑world use
