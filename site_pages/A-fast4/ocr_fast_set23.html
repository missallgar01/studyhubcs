1. Explain

[1]

One benefit of a real-time operating system.

Answer: • Provides guaranteed response times to events within a specified deadline
• Suitable for safety‑critical applications such as medical devices, automotive systems and industrial control
• Schedules tasks so that high‑priority, time‑sensitive tasks are serviced predictably
• Helps ensure system reliability by avoiding unacceptable delays

2. Identify

[1]

One reason why recursion uses more memory

Answer: • Each recursive call creates a new stack frame on the call stack
• Stack frames store return addresses, parameters and local variables for that call
• Deep recursion leads to many active stack frames at once
• This can cause high memory usage and potentially a stack overflow if the recursion is too deep

3. Explain

[1]

One disadvantage of recursion.

Answer: • Each recursive call uses a new stack frame, which can consume significant memory
• Deep or uncontrolled recursion risks a stack overflow error
• May be slower than an equivalent iterative solution due to call overhead
• Can be harder to trace and debug because control flow jumps between many call levels

4. Describe

[1]

One benefit of automation.

Answer: • Allows repetitive tasks to be performed consistently and quickly with minimal human intervention
• Reduces the chance of human error in routine processes
• Can increase productivity and throughput in both software and industrial contexts
• Frees up human workers to focus on more complex or creative tasks
