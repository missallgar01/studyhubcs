FAST FOUR STARTER

1. Explain

[1]

Why virtual memory is needed.

Answer: • Physical RAM is limited, but programs may require more memory than is physically available
• Allows many programs to appear to have their own large, contiguous address space
• Enables multitasking without each process having to fit entirely in RAM
• Provides memory protection so faulty processes are less likely to corrupt others

2. Identify

[1]

One benefit of open-source software.

Answer: • Source code is available for inspection, modification and redistribution
• Encourages collaboration and peer review, which can lead to rapid improvement and bug fixing
• Reduces vendor lock‑in because users are not tied to a single company
• Often available at low or no cost, making it accessible to more users

3. Identify

[1]

One benefit of using SSDs over HDDs.

Answer: • Shorter boot and load times due to higher data transfer rates
• No mechanical latency or seek time because data is accessed electronically
• Quieter operation and reduced power usage
• Lower failure rates from mechanical wear compared with spinning disks

4. Explain

[1]

One drawback of object-oriented programming.

Answer: • Abstraction layers, dynamic dispatch and indirection can introduce performance overhead
• Poorly designed class hierarchies can become deep and hard to maintain
• Can be more difficult for beginners to understand than simple procedural code
• Not always the best fit for very small programs or highly performance‑critical tasks
