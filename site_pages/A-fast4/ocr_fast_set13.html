One benefit of object-oriented programming.

Answer: • Encourages encapsulation, keeping data and the methods that operate on it together in classes
• Supports inheritance, allowing common behaviour to be defined once and reused in subclasses
• Enables polymorphism, so code can work with objects of different types through a common interface
• Makes large systems easier to organise and extend by modelling them as interacting objects

2. Identify

[1]

One benefit of modular programming.

Answer: • Breaks a large program into smaller, self‑contained modules that are easier to understand
• Allows different programmers to work on separate modules in parallel
• Encourages code reuse, as modules can be used in other programs
• Makes testing and debugging easier because faults can be isolated to individual modules

[1]

One difference between RISC and CISC

Answer: • RISC uses a small set of simple instructions, each designed to execute in one clock cycle
• CISC uses a large set of more complex instructions that may take multiple cycles
• RISC architectures rely heavily on registers and simple addressing modes
• CISC architectures include instructions that can perform multi‑step operations or complex memory access patterns

One benefit of abstraction.

Answer: • Hides unnecessary implementation details so programmers can focus on the essential features of a system
• Reduces complexity by presenting a simpler interface to underlying code or hardware
• Makes systems easier to maintain because implementation can change without affecting users of the abstraction
• Encourages modular design and reuse of components
