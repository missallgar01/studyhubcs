1. Identify

[1]

One drawback of binary search.

Answer: • Requires the data to be sorted before searching, which has its own cost
• Maintaining a sorted data structure can be expensive when inserting or deleting items
• Harder to implement correctly than simple linear search
• Not suitable for data structures that do not support efficient random access

2. Describe

[1]

One drawback of using abstraction in design.

Answer: • Excessive abstraction can hide important implementation details from developers
• Too many abstraction layers can make systems difficult to trace and debug
• Poorly chosen abstractions may be hard to change later without affecting many components
• Abstraction layers can introduce extra overhead, slightly reducing performance

3. Describe

[1]

One benefit of using logic gates in hardware design.

Answer: • Allow Boolean logic to be implemented physically using electronic circuits
• Form the basic building blocks for more complex components such as adders, multiplexers and flip‑flops
• Enable designers to create predictable, digital systems from simple combinations
• Support reliable, binary operation that is resistant to small electrical noise

4. Identify

[1]

One reason why recursion uses more memory.

Answer: • Each recursive call creates a new stack frame on the call stack
• Stack frames store return addresses, parameters and local variables for that call
• Deep recursion leads to many active stack frames at once
• This can cause high memory usage and potentially a stack overflow if the recursion is too deep
