FAST FOUR STARTER

1. Identify

[1]

One ethical issue with AI.

Answer: • AI systems may learn and reinforce existing biases in training data, leading to unfair or discriminatory decisions
• Decisions made by complex AI models can lack transparency, making it hard to explain or challenge outcomes
• Use of AI for surveillance can threaten privacy and civil liberties
• Automation driven by AI may displace jobs, raising social and economic concerns

2. Identify

[1]

One disadvantage of logic gates.

Answer: • Complex circuits require many gates, increasing cost, power usage and heat
• Physical limitations such as propagation delay restrict maximum speed
• Any single gate failure can affect the correctness of larger digital systems
• Designing and testing large gate‑level circuits is time‑consuming

3. State

[1]

The purpose of a compiler

Answer: • Translates high‑level source code into machine code or an intermediate object code
• Performs lexical, syntax and semantic analysis to detect errors
• Applies optimisations to improve the performance and efficiency of the final executable
• Produces an executable file that can run independently of the compiler

4. Describe

[1]

How an operating system schedules processes.

Answer: • Uses a scheduling algorithm (e.g. Round Robin, priority, FCFS) to choose which process runs next
• Allocates CPU time slices to each ready process
• Performs context switches, saving and restoring process state in registers and memory
• Aims to maximise CPU utilisation while maintaining fairness and responsiveness
• May adjust priorities dynamically (e.g. aging) to prevent starvation
