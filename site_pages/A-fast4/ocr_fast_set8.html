1. Explain

One drawback of using encryption.

Answer: • Encrypting and decrypting data adds computational overhead and can slow systems down
• Key management is complex; lost or stolen keys can render data inaccessible or insecure
• Encrypted data is harder to inspect for debugging, monitoring or filtering
• Strong encryption can complicate lawful access or digital forensics

2. State

[1]

One benefit of using cache memory.

Answer: • Stores copies of frequently accessed data closer to the CPU, reducing access time
• Decreases the average time to read from or write to main memory
• Improves performance for programs with good locality of reference (repeated access to nearby locations)
• Helps keep the CPU busy rather than waiting on slower RAM

3. State

[1]

The purpose of the Program Counter (PC)

Answer: • Holds the memory address of the next instruction to be fetched
• Automatically increments after each fetch to keep program flow sequential
• Can be changed by branch/jump instructions to alter the flow of control
• Allows the CPU to track where it is in the current program

4. Describe

[1]

One benefit of scheduling algorithms.

Answer: • Improve overall CPU utilisation by reducing idle time
• Help provide fairness so no process is permanently starved of CPU time
• Can reduce average waiting and turnaround times for processes
• Allow different priorities to be enforced for different types of jobs
