FAST FOUR STARTER

1. State

[1]

One use of ROM

Answer: • Stores firmware or microcode that should not change during normal operation
• Holds the bootstrap program used when the computer is powered on
• Provides permanent storage for device configuration or lookup tables
• Retains its contents even when power is switched off

2. State

[1]

One drawback of pipelining.

Answer: • Hazards such as data, control and structural hazards can cause pipeline stalls or bubbles
• Requires additional hardware such as forwarding logic and hazard detection units
• Branch instructions can reduce efficiency due to mispredictions and flushing of the pipeline
• Increases the complexity of the CPU design

3. Explain

[1]

One benefit of using virtualisation.

Answer: • Allows multiple virtual machines to share a single physical server, maximising hardware utilisation
• Provides isolation between virtual machines, so a crash in one does not affect others
• Simplifies backup, migration and recovery using snapshots and images
• Makes it easier to test software on different operating systems without extra hardware

4. Identify

[1]

One benefit of using Unicode.

Answer: • Supports a very large range of characters from many writing systems worldwide
• Allows text in multiple languages to be stored and processed consistently
• Reduces compatibility problems caused by different local character sets
• Makes internationalisation and localisation of software easier
