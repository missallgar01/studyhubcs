One drawback of floating-point representation.

Answer: • Many decimal fractions cannot be represented exactly, leading to rounding errors
• Arithmetic operations can accumulate rounding errors over time
• Comparisons between floating‑point values can be unreliable if precision is not considered
• Implementation is more complex and slower than integer arithmetic

[1]

One benefit of recursion over iteration.

Answer: • Expresses some algorithms in a clearer and more concise way than loops
• Makes code for recursive data structures (like trees) easier to understand
• Avoids manual management of stack data structures in the program itself
• Can make correctness proofs easier using mathematical induction

3. State

[1]

One disadvantage of virtualisation.

Answer: • Introduces overhead because multiple virtual machines share the same physical resources
• Performance may be lower than running directly on bare metal hardware
• Requires careful configuration to avoid resource contention between virtual machines
• Adds complexity to system management and security

[1]

The purpose of a compiler.

Answer: • Translates high‑level source code into machine code or an intermediate object code
• Performs lexical, syntax and semantic analysis to detect errors
• Applies optimisations to improve the performance and efficiency of the final executable
• Produces an executable file that can run independently of the compiler
