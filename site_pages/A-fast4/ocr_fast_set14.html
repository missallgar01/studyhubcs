FAST FOUR STARTER

1. Identify

[1]

One advantage of using cache memory.

Answer: • Provides much faster access times than main memory (RAM)
• Stores recently used instructions and data so repeated accesses are quicker
• Reduces the average time the CPU must wait for memory accesses
• Improves performance of loops and frequently executed code sections

2. Describe

[1]

One disadvantage of using interrupts.

Answer: • Frequent interrupts can lead to high context‑switching overhead and reduced performance
• Interrupt handling code (ISRs) must be carefully written to avoid race conditions and deadlocks
• Debugging interrupt‑driven systems can be more difficult than debugging simple polling loops
• Prioritising interrupts incorrectly may cause important tasks to be delayed

3. Describe

[1]

What a stack is used for

Answer: • Stores return addresses when subroutines and functions are called
• Holds local variables and parameters for active procedures
• Keeps track of nested and recursive function calls via stack frames
• Provides a last‑in, first‑out (LIFO) structure for temporary data storage
• Supports evaluation of arithmetic expressions and undo operations

4. Describe

[1]

One benefit of using recursion.

Answer: • Encourages a divide‑and‑conquer approach by breaking problems into smaller subproblems
• Simplifies code for algorithms that are naturally recursive, such as tree traversal and quicksort
• Reduces the need for explicitly managed stacks in the source code
• Allows elegant definitions for mathematical functions like factorial
