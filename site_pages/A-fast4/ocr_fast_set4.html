One benefit of pipelining in CPUs.

Answer: • Overlaps the execution of multiple instructions, increasing instruction throughput
• Allows the CPU to start fetching a new instruction before the previous one has finished executing
• Can significantly improve performance without increasing the clock speed
• Makes better use of the different functional units within the processor

2. Explain

[1]

One ethical drawback of AI systems.

Answer: • AI decision‑making can be opaque, making it hard for affected individuals to understand or challenge outcomes
• Biased training data can lead to unfair treatment of particular groups
• Widespread automation may displace workers and widen social inequality
• Use of AI in surveillance or profiling can undermine privacy and civil liberties

3. Identify

[1]

One advantage of using cache memory

Answer: • Provides much faster access times than main memory (RAM)
• Stores recently used instructions and data so repeated accesses are quicker
• Reduces the average time the CPU must wait for memory accesses
• Improves performance of loops and frequently executed code sections

4. Describe

[1]

One benefit of using interrupts.

Answer: • Allows the CPU to handle asynchronous events efficiently without constant polling
• Improves responsiveness to external devices such as keyboards, disks and timers
• Enables time‑critical tasks to interrupt less important work when necessary
• Helps implement multitasking by pre‑empting running processes
