1. Explain

[1]

One drawback of using SSDs.

Answer: • More expensive per gigabyte than traditional hard disk drives
• Flash memory cells have a limited number of write cycles, leading to wear over time
• Data recovery from failed SSDs can be difficult or impossible
• Performance can degrade as the drive becomes full if not managed carefully

2. Explain

[1]

The difference between a compiler and an interpreter

Answer: • A compiler translates the entire source code into machine code before execution, producing an executable file
• An interpreter translates and executes the program line by line at run time without producing a separate executable
• Compiled programs generally run faster because translation happens once
• Interpreted programs are easier to test and debug because errors are reported as soon as a problematic line is reached

3. Describe

[1]

One benefit of using floating-point representation.

Answer: • Can represent a very wide range of real numbers, including very large and very small values
• Supports fractional values for scientific and engineering calculations
• Uses a mantissa and exponent to provide approximate values with controlled precision
• More suitable than fixed‑point or integer representation for many numerical algorithms

4. State

[1]

One disadvantage of real-time operating systems.

Answer: • Often require specialised hardware and careful system tuning, increasing cost
• Feature sets may be limited to maintain strict timing guarantees
• Development and testing must be rigorous to prove deadlines are always met
• Missing a deadline can have serious safety or financial consequences
