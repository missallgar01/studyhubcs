2. Explain

[1]

How interrupts improve CPU performance.

Answer: • Remove the need for constant polling loops that repeatedly check devices
• Allow the CPU to perform useful work until a device signals it needs attention
• Provide fast response to urgent events such as I/O completion or timer expiry
• Enable pre‑emptive multitasking by interrupting a running process when needed
• Help achieve higher overall CPU utilisation
