Software & Operating Systems – Notes

HTML — Structure of Web Pages

HTML defines the structure and order of content on a webpage. It tells the browser what each piece of content is, not how it should look.

Example: A simple HTML page 
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
    <h1>Welcome</h1>
    <p>This is my webpage</p>
  </body>
</html>

HTML Elements — Images, Links, Lists

Images use the <img> tag:

<img src="logo.png" width="200" height="80">

Links use the anchor tag:

<a href="https://www.example.com">Visit Example</a>

Lists:

<ul>
  <li>Computer Science</li>
  <li>Maths</li>
</ul>

CSS — Styling Web Pages

CSS controls the appearance of HTML elements. It separates presentation from content.

Example: External CSS file 
body {
  background-color: white;
  font-family: Arial;
}

h1 {
  color: blue;
}

This CSS will apply to every page that links to the stylesheet.

Classes and IDs (CSS Selectors)

Classes can be reused. IDs must be unique.

Example: 
/* CSS */
.highlight {
  background-color: yellow;
}

#main {
  border: 2px solid black;
}
        
<div id="main">
  <p class="highlight">Important text</p>
</div>

JavaScript — Adding Interactivity

JavaScript is interpreted by the browser and allows pages to respond to user actions.

Example: Change text when a button is clicked 
<button onclick="changeText()">Click me</button>
<p id="msg">Hello</p>

<script>
function changeText() {
  document.getElementById("msg").innerHTML = "Goodbye";
}
</script>

Why JavaScript is Useful

  • Checks form input before sending data to the server
  • Reduces server load
  • Improves user experience
Example: Simple validation 
if (password.length < 8) {
  alert("Password too short");
}

Search Engine Indexing

Search engines use web crawlers to collect information about pages.

Example:

A crawler visits Page A, follows links to Pages B and C, extracts keywords like “computer science”, and stores them in an index.

PageRank Algorithm (Worked Concept Example)

PageRank measures importance based on incoming links.

Example:

If Page C is linked by two pages, and one of those pages is already highly ranked, Page C’s rank will increase more than if both links came from low-ranked pages. 

PR(C) = (1 − 0.85) + 0.85 × (PR(B)/2 + PR(D)/3)

Server-Side Processing — Example

Data is sent to a server, processed, and a response is returned.

Example:

A login form sends a username and password to a server. The server checks the database and responds with success or failure.

Client-Side Processing — Example

Processing happens on the user’s device.

Example:

A form checks that an email address contains “@” using JavaScript before allowing submission.

Exam Tip

Always link examples to advantages:

  • Client-side → fast response, reduced server load
  • Server-side → security, large processing tasks
