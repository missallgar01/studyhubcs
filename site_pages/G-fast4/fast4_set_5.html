3. Explain

[2]

Explain one disadvantage of using a star network topology.

Answer: • A linked explanation such as:
• All communication could fail because/if the central device fails
• It can be difficult/expensive to set up because each device needs a cable to connect to the central device
• The number of devices that can be connected is limited because the central device supports a fixed number of connections

4. Explain

[2]

Explain one benefit of using subprograms.

Answer: • An explanation to include two from:
• The subprogram may be used more than once in a program so that writing, debugging, testing will save time
• The subprogram performs one specific/contained task so it can be moved away from the main program code
• Subprograms for common tasks can be stored in libraries and reused in other programs so that they don’t have to be re-written
