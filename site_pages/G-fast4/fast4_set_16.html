Explain the reason why at least nine bits are needed to store 300 different binary patterns.

Answer: • One from:
• 2⁹ gives 512 patterns whereas 2⁸ gives 256 patterns
• 2⁸ does not give enough patterns whereas 2⁹ gives more than enough patterns

3. Explain

[2]

A company needs to secure its network from attacks by its employees. Explain the best choice of penetration testing the company should use.

Answer: • An explanation that includes the ideas:
• Permission is given to a person to carry out investigation of security issues in the network
• White-box / white-hat because system information is provided to the tester
• The company should give a security expert an ID and password and details of how the network is configured , and ask them to try to find vulnerabilities that would enable an employee to access, delete and/or leak sensitive data or install malicious software

4. Explain

[2]

A factory has two file servers installed in an office. A closed-circuit television system monitors the factory. Explain one other method of physical security that could be used to protect the servers.

Answer: • (Digital/biometric) locks / magnetic card scanners / security personnel could be put on the office door to prevent unauthorised access
• Movement sensors , to activate an alarm/light
• Security lights/alarms , so that intruders would be scared off/identified
• Alarms/alerts could trigger because there is unauthorised access
• The servers/assets could have security tags , so that the owner can be tracked if stolen
• ID cards could be issued to all staff , to prevent impersonation/to allow unknown people to be challenged
• Warning posters could be positioned at entrances , to make people aware of shouldering/tailgating/two people entering together
