FAST FOUR STARTER

1. State/Identify

[2]

State two ways that a user can increase the useful life of a smartphone rather than throwing it away.

Answer: • One mark each for a maximum of 2:
• Use it as long as possible / don’t replace it
• Sell/give it to a company that will recondition it
• Give it away to a friend or charity
• Repair it if it breaks
• Keep the software updated

2. State/Identify

[1]

State the name of an email protocol.

Answer: POP3, IMAP or SMTP

3. Describe

[4]

Describe two ways that a high-level language differs from a low-level language.

Answer: • High-level languages use instructions that look like English , whereas a low-level language uses mnemonics/binary code
• A high-level language statement generates many lines of machine code , whereas each line of low-level languages is/generates a single machine instruction
• High-level languages are general purpose/exist across microprocessors/CPUs/machine-independent , whereas a low-level language is microprocessor/CPU/machine specific
• High-level languages are abstracted from the hardware , whereas low-level languages manipulate the hardware directly

The ASCII code 65 represents the letter A. Give the letter represented by the ASCII code 68.
