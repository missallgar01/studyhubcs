FAST FOUR STARTER

1. Characteristics/Features

[2]

An embedded computer system is part of a larger machine. State two characteristics of embedded systems that make them different from general purpose computers.

Answer: • Limited processing resources
• Low power consumption
• Simple/robust operating system
• Low cost per unit
• Smaller in size
• Less storage
• Limited user interface

2. Explain

[2]

Explain one benefit to a user of using IMAP to access emails.

Answer: • Emails can be accessed from multiple devices because IMAP does not remove/delete emails from the server
• Emails can be accessed through a browser because IMAP doesn’t need a client application to download messages

3. State/Identify

Answer: • One mark for the benefit:
