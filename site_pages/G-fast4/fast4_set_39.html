2. Advantages/Disadvantages

[2]

A team of developers is using an audit trail when working on a program. State two advantages of keeping an audit trail.

Answer: • One mark for each to a maximum of 2:
• There’s a record of who changed what when
• Identifies the point at which an error/security issue was introduced
• Enables the program to be rolled back to a previous state
• Improve accountability/produces robust code

3. Describe

[2]

Describe one difference between a syntax error and a logic error.

Answer: • A syntax error is caused by using the grammar/form/words of the programming language incorrectly , whereas a logic error is caused by an error in the design of the algorithm
• A syntax error causes the program not to interpret/compile/translate, whereas a logic error produces an incorrect/wrong result

4. Explain

[2]

Explain one reason why software should be patched regularly.

Answer: • An explanation to include two from:It helps prevent unauthorised access to data
• because security issues could be missed in development
• because there could be interactions with other software
• because a malicious user could have published a vulnerability on the web
• because sometimes vulnerabilities only become apparent after code has been released to the public
