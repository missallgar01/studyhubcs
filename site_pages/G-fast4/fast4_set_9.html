FAST FOUR STARTER

1. Describe

[2]

Describe one way that changing the number of bits allocated to the colour depth determines how an image is represented.

Answer: • A description to include two from:
• Increasing the number of bits used for each pixel allows more colours to be represented in each pixel OR
• Increasing the number of bits allows more pixels to be used providing a greater level of resolution OR
• Decreasing the number of bits used for each pixel allows fewer colours to be represented in each pixel OR
• Decreasing the number of bits allows fewer pixels to be used providing a reduced level of resolution

2. Construct

[2]

Construct an expression to convert 40 681 930 227 712 bytes to tebibytes.

Answer: • Sight of 1024⁴ anywhere for number of bytes in a tebibyte
• 40 681 930 227 712 / 1024⁴
