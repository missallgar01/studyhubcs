FAST FOUR STARTER

1. State/Identify

[1]

Identify an additional piece of software that is required for translating assembly code

Answer: Assembler

2. State/Identify

[1]

State the name of the high-level programming language translator that executes a line of code immediately after translating it.

Answer: Interpreter

3. Describe

[2]

Describe how an operating system organises files and folders.

Answer: • Uses a hierarchical/tree structure with a root node/directory
• Each position on the tree/path is a directory/sub-directory/folder, or file
• Each file has a unique path in relation to the root

4. State/Identify

[1]

Identify the hardware component that carries instructions from memory to the CPU.

Answer: Data Bus
