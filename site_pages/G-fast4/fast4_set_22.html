FAST FOUR STARTER

1. Explain

[2]

Explain the best choice of secondary storage for the battery-operated mower.

Answer: The mower should use solid-state secondary storage because it has no moving parts / can use very little power from the battery / is robust to bumps and knocks

2. State/Identify

[1]

State the type of secondary storage that stores data as electric charges.

Answer: SSD/ Solid State/ Flash

3. Describe

[2]

Describe how the link layer of the TCP/IP protocol stack works.

Answer: • It converts the data into a suitable signal/electrical/radio for transmission on the media (Ethernet cable/Wi-Fi)
• It uses/adds a physical/MAC address for a source/destination
• Receives data from the network layer to send via physical hardware

4. State/Identify

[2]

Algorithmic bias could be caused by using inaccurate data to train a machine learning algorithm. State two other ways that the data used to train an algorithm could cause algorithmic bias.

Answer: • One mark each for a maximum of 2:
• Not enough / insufficient data
• Biased / prejudiced data / not representative
• Non-aspirational data
