1. Describe

[2]

One function of an operating system is user management. Describe one purpose of user management.

Answer: • A description to include:
• To provide access to the device by controlling who can log on to the device
• To allow a user to log on by checking the username/password
• By authenticating the user before allowing them to log on

2. Describe

[2]

Describe how a firewall protects a local area network (LAN).

Answer: • A description to include two from:
• Monitors/checks traffic using a set of rules / list of authorised or unauthorised computers, addresses, or protocols to decide if data is allowed into or out of the network

3. Reason

[1]

Give one reason the performance of a solid-state drive is not affected by fragmentation.

Answer: • A solid-state device has no moving parts
• Seek time is not affected by scattered data because it is read electronically

4. Describe

Describe how data is stored on solid state storage

Answer: • A description to include two from:
• Data is stored using electronic circuits / transistors
• Electrons are trapped or released in a floating gate / memory cell
• The presence or absence of charge represents binary 1s and 0s
