FAST FOUR STARTER

1. Characteristics/Features

[3]

When using a compiler to translate source code to machine code, the translation only needs to be done once. Give three other features of a compiler.

Answer: • Translates the entire source file to machine code in one go / translates all the source code prior to execution
• Shows all syntax errors at the end of the translation process
• Produces a single executable/object code/platform-dependent output
• Separates the tasks of translation and execution

2. Describe

[2]

Computer worms are a type of malware. Describe one way that a worm can move from machine to machine.

Answer: • Worms make copies of themselves and send them to everyone in the computer’s email contacts list / send them via previous connections used by the computer
• Worms can spread (themselves) through emails and do not need a user to unknowingly distribute/execute them
• Worms self-replicate and spread without the need for a host file

4. State/Identify

[4]

The Internet of Things has allowed devices with embedded systems to function independently and to collect and exchange data without the need for humans. One such device is a battery-powered mower for the garden. (i) Give one ways each category could be used by the mower. Connectivity Input Process Output

Answer: • Answer
One mark for each to a maximum of 4, with at least one in each category Connectivity:
• The mower could connect to a wireless/wired network
• The mower could have a GPS device . Input:
• A phone/desktop/software application could be used to schedule when it will work
• A map of the garden would be stored on the mower
• The mower would have sensors to know if it ran into things / if it was dark / if it were raining / if it ran outside the garden. Process:
• The microcontroller in the mower would run a program
• A map of the garden would be read by the program so the mower would know where to go
• The program would monitor the power status to know if it needed recharging. Output:
• The program on the mower could upload statistics/information back to the application
• The mower could have a digital readout / flashing lights to indicate status
