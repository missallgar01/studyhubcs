3. Describe

[2]

Algorithms use arrays and records to hold data. Describe a record.

Answer: • A list / data structure / collection of elements/fields/values where each element/field/value may be a different data type / is related to the others For both marks, the expansion must associate with the statement

4. Explain

[2]

Some users are given administrator privileges. Explain one way an operating system allows an administrator to manage users.

Answer: • A linked explanation such as:
• Users can be added/deleted so multiple people can use the same computer
• Edit user permissions so only specific users can securely access their storage space
• Control the amount of resources/storage each user can access so the limited storage on the machine can be shared
• Enforce user permissions so only certain users are allowed to install programs / access certain files
