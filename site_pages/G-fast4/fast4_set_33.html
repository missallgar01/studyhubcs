Explain a runtime error.

Answer: • The program crashes/stops because the operation the computer is asked to do is impossible
• The program crashes because the CPU cannot execute one of the instructions in the code For both marks, the expansion must follow/associate with the statement.
Do not award credit for an example in isolation such as division by zero.
Be careful not to award marks for syntax/logic error
