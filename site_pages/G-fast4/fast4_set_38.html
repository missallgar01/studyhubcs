In arithmetic, subtraction can be done by adding a negative number. Calculate 18 − 8, using 8-bit binary and two's complement. Convert the result back to denary. Show all your working.

Answer: • 18 converted to 8-bit binary 0001 0010
• −8 converted to two’s complement 8-bit binary 1111 1000
• Addition performed correctly (1 mark for each nibble)○ 8-bit binary result of 0000 1010
