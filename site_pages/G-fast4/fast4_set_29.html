3. Give

[2]

Give two disadvantages of a bus network topology.

Answer: • The whole network stops working if the backbone/cable is broken
• The number of collisions increases / the network will slow down as the number of devices/users increases
• A greater security risk (because all the nodes on the network see all the data packets)
• The network stops working if the terminators are missing/not working
• Only one device can transmit data at a time
