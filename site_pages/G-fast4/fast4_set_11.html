2. Describe

[4]

Describe two ways a compiler differs from an interpreter.

Answer: • Two linked descriptions from:
• A compiler carries out translation once prior to execution whereas an interpreter carries out translation every time the program executes
• A compiler produces a stand-alone executable file whereas an interpreter is required each time the code is run
• A compiler reports errors after translation is complete whereas an interpreter reports errors as they occur
