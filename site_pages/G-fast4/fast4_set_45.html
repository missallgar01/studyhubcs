2. Characteristics/Features

[2]

A compiler translates source code to machine code. If the source code is edited, it must be recompiled. Give two other characteristics of a compiler.

Answer: Award one mark for any of the following up to a maximum of two marks: • Translates the whole program in one go (1) • Errors are reported at the end of translation (1) • The executable/machine code is architecture specific (1)
