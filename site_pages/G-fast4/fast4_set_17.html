Give two disadvantages of a bus network topology.

Answer: Award one mark for any of the following, up to a maximum of two marks:• The whole network stops working if the backbone/cable is broken (1)• The number of collisions increases / the network will slow down as the number of devices/users increases (1)• A greater security risk (because all the nodes on the network see all the data packets) (1)• The network stops working if the terminators are missing/not working (1)• Only one device can transmit data at a time (1)
