FAST FOUR STARTER

1. Give

[2]

Devices are connected to networks. (i) Give two reasons for connecting computers in a network.

Answer: • Can share files/data
• Can share applications/software
• Can collaborate
• Can share peripherals (printers, speakers)
• Can share connectivity (Internet connection, hotspot)
• Can access files from any computer / hot desk
• Central control/management (security, software updates, backup, remote support, users, remote monitoring)
• Allows communication (email, social media, video conferencing, online meeting, collaborative games)

2. Construct

[4]

Construct an expression to calculate the transmission rate, in megabits per second, required to transmit a 1.4 gibibyte file in 13 minutes. You do not need to do the calculation.

Answer: • 13 × 60 or 780 (seconds)
• 1.4 × 8 × 1024³
• 1000²
• Fully correct response Example: (1.4 × 8 × 1024³) ——————————— (13 × 60 × 1000²)

3. Describe

[1]

A code review is carried out by a programmer or an automated system and cost money. What is another disadvantage of a code review by a automated system

Answer: May not identify all issues (that it is not programmed to find) May only find well known problems

4. Define

[2]

Define the term 'subprogram'.

Answer: • One mark each for a maximum of 2:
• A subprogram is a self-contained block of code
• A subprogram performs a specific/dedicated task
• It can be ‘called’ by the main program or other subprograms, when it is needed
