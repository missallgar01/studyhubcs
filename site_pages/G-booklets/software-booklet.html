Systems Software

Systems software is software designed to maintain and control the hardware. It also provides an interface between the hardware and software.

Operating System

An operating system controls the hardware and software and provides an interface for the user.

Utility Programs

Utility programs help to maintain the computer.

Application Software

Application software is a collection of one or more programs used to solve a specific task.

Types of Application Software

Examples of application software include word processing software, spreadsheet software, database software, education software, and entertainment software.

Compression Software

Compression software reduces the size of a file stored on secondary storage. Smaller files are easier to transmit across a network as they require fewer packets and take up less storage space.

Defragmentation Software

Defragmentation software takes fragmented files and rearranges the segments so that they run contiguously. This decreases read and write time and speeds up computer performance.

Other Utility Software

Other utility software includes automatic backup and system updates, anti-virus and firewall software, and disk repair tools used to fix corrupted files.

User Interface

A user interface allows the user to communicate with the hardware. Types of user interface include command line interfaces, graphical user interfaces (GUI), and voice interfaces.

Memory Management

Memory management uses paging to allocate sections of RAM to processes. Data is moved between RAM and virtual memory to free space and allow higher-priority processes to run efficiently.

Peripheral Management and Drivers

A device driver is a program that controls a peripheral device and allows the operating system to communicate with it. A peripheral is an external device such as a monitor or keyboard.

User Management

User management allows users to log on by checking usernames and passwords. Users can be added or deleted, permissions can be enforced, and the amount of storage each user can access can be controlled.

File Management

File management stores data in a hierarchical structure and allows users to create, rename, copy, move, delete, search for, retrieve, and edit files and folders.

Process Scheduling

A multitasking operating system executes one process at a time but allows multiple applications to run by rapidly switching between processes using time slices. Processes are held in a queue and controlled by a scheduling algorithm.

Authentication

Authentication is a process used to test that a person is who they claim to be.

Code Reviews

Code reviews involve checking program code with another person to identify errors or security issues that may have been missed.

Audit Trails

An audit trail records changes made to software, including who made the change and when. This allows problems or security flaws to be traced and fixed.

Check your understanding

One function of an operating system is to manage processes.
(i) Describe one way the operating system makes sure each process can use the CPU. (2)
(ii) Give one other function of an operating system. (1)
(Total for question = 3 marks)

How to answer (DESCRIBE / GIVE):
  • Describe = what happens, step or mechanism
  • Use keywords such as scheduler, time slice, queue
  • For part (ii), just name one function
Mark scheme – ALL acceptable answers:
  • Scheduler controls processes (1)
  • Each process gets a time slice (1)
  • Processes held in a queue (1)
  • Priority ordering (1)
  • Incomplete processes moved to back of queue (1)
  • Other OS functions:
    • File management (1)
    • Peripheral management (1)
    • User management (1)
    • User interface (1)
Model full-marks answer:

The operating system uses a scheduler to allocate a short time slice of CPU time to each process. This ensures all processes get a chance to run. Another function of the operating system is file management.

One role of the operating system is to allocate memory to processes.
Describe how the operating system makes use of virtual memory when there is a shortage of main memory.
(2)
(Total for question = 2 marks)

Command word focus: DESCRIBE
  • Explain what happens to inactive processes
  • Explain how they return to RAM when needed
Mark scheme – ALL acceptable answers:
  • Processes/programs/instructions/data not using the CPU are moved out of RAM/main memory (1)
  • Moved into secondary storage / hard disk / SSD (1)
  • When needed again, they are swapped back into RAM/main memory (1)
  • RAM/main memory is divided into pages (1)
  • A paging algorithm swaps pages between RAM and secondary storage (1)
Model full-marks answer:

When RAM is low, the operating system moves inactive processes from RAM to secondary storage. When the process is needed again, it is swapped back into RAM.

Backups are used to protect data.
(i) State one reason why backups are needed. (1)
(ii) Explain one disadvantage of backing up data off-site. (2)
(Total for question = 3 marks)

Command word focus: STATE / EXPLAIN
  • State = short reason
  • Explain = disadvantage + effect
Mark scheme – ALL acceptable answers:
  • (i)
    • Recover data after hardware failure (1)
    • Recover data after malware attack (1)
    • Recover deleted/corrupted files (1)
    • Disaster recovery (1)
  • (ii)
    • Data transfer can be slow (1) because it depends on internet speed (1)
    • Security risks during transmission (1)
    • Ongoing cost of off-site storage (1)
    • Reliance on third-party provider (1)
Model full-marks answer:

Backups are needed to recover data after a hardware failure. Off-site backups can be slower because data must be transferred over the internet.
