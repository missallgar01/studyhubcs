CPU, Buses, Fetch–Decode–Execute Cycle & Embedded Systems

Recap

Main CPU Components

  • Control Unit (CU) – decodes instructions and sends signals to control how data moves around the CPU and memory.
  • Arithmetic Logic Unit (ALU) – carries out calculations and logical comparisons (e.g. greater than, less than).
  • Clock – a signal to synchronise tasks. A clock cycle is known as a tick. Each cycle has a high state and a low state. For each tick of the clock the CPU can process one or more instructions. The more clock ticks there are, the faster instructions can be processed.

CPU Registers (temporary stores for instructions being processed)

  • Program Counter (PC) – keeps track of where the CPU is in the program, points to the next instruction in the cycle.
  • Memory Address Register (MAR) – holds the address of the instruction to be fetched.
  • Memory Data Register (MDR) – stores the instruction about to be executed.
  • Accumulator (ACC) – stores the most recent result of processing.

Buses

  • Address bus – carries memory addresses, uni-directional.
  • Data bus – carries instructions and data items, bi-directional.
  • Control bus – carries commands from the CPU and status messages of devices, bi-directional.

Fetch–Decode–Execute (FDE) Cycle

The fetch–decode–execute cycle is the process of fetching instructions from memory, decoding the instructions and executing them.

6. Fetching an Instruction from Memory

Describe the role of the control unit, the control bus, the data bus and the address bus when fetching an instruction from memory.

(4 marks)

Answer: The original booklet did not include a fully written model answer for this question. A typical answer would explain, for example, that the control unit places the address of the next instruction in the MAR, the address bus carries this address to memory, the control bus sends a read signal, and the data bus carries the instruction back to the CPU.

8. Data Bus

Describe one reason the data bus needs to be bi-directional.

Answer: The CPU needs to be able to read program instructions and data from memory and write the results of operations to memory. The CPU consists of registers (which are temporary stores for instructions being worked on), the cache, the Control Unit and the ALU.

9. Clock Speed

State the reason why a higher clock speed is desirable.

(1 mark)

Answer: More instructions can be carried out per second.
