A linear search algorithm can be used on both a sorted and an unsorted array.
Describe how a linear search algorithm operates on an unsorted array.
(4)
(Total for question = 4 marks)

Command word focus: DESCRIBE
  • Explain step-by-step
  • Include start, comparison, and end conditions
Mark scheme – ALL acceptable answers:
  • Starts at the first element in the array (1)
  • Compares each item to the target value (1)
  • Moves to the next item if not found (1)
  • Stops when item is found or end of array is reached (1)
  • Reports whether the item was found or not (1)
Model full-marks answer:

A linear search starts at the first item in the array and compares each value to the target. If the value does not match, it moves to the next item. The search stops when the item is found or the end of the array is reached.

This algorithm searches a sorted array of numbers for a target value. The target value may or may not be in the array. The use of the found variable helps to make the algorithm efficient.
Describe how the use of the passed variable also helps to make the algorithm efficient.
(2)
(Total for question = 2 marks)

Command word focus: DESCRIBE
  • Explain what happens when value is passed
  • Link to stopping early
Mark scheme – ALL acceptable answers:
  • Passed indicates the target cannot be in the array (1)
  • Stops further comparisons (1)
  • Prevents unnecessary iterations (1)
Model full-marks answer:

The passed variable shows when the target value has been passed in the sorted array. This allows the algorithm to stop early, reducing unnecessary comparisons.

The efficiency of algorithms can be analysed.
(i) Explain the worst case for a linear search on an array. (2)
(ii) Explain the best case for a bubble sort on an array. (2)
(Total for question = 4 marks)

Command word focus: EXPLAIN
  • Link comparisons to efficiency
  • Describe when fewest/most comparisons occur
Mark scheme – ALL acceptable answers:
  • (i) Linear search worst case:
    • Target is last item or not in array (1)
    • Every item must be compared (1)
  • (ii) Bubble sort best case:
    • Array already sorted (1)
    • No swaps needed / minimal comparisons (1)
Model full-marks answer:

The worst case for a linear search is when the item is last or not present, so every value must be checked. The best case for a bubble sort is when the array is already sorted, so no swaps are needed.

The Airport
An international airport uses a variety of computer systems to deal with passengers, flights and employees.

When an item of luggage leaves the check-in area, it is sorted and delivered to the aircraft.
Luggage is sent to either the international or the domestic sorting hub.

If the label begins with 'I', the item is sent to the international hub. If the label begins with 'D', the item is sent to the domestic hub.
If the label is damaged and not readable, the item is sent for manual inspection.

Draw a flowchart to represent this process for a single item of luggage.
Put a line through any work you do not want to be marked.
(6)

Command word focus: DRAW / REPRESENT
  • Use correct flowchart symbols
  • Include all decision paths
  • Ensure every path ends
Mark scheme – ALL acceptable points:
  • Maximum 3 marks for functionality
  • Maximum 3 marks for accuracy of notation
  • Start symbol included
  • Reads first letter of label
  • Decision: letter = 'D'
  • If yes → send to domestic hub → stop
  • If no → decision: letter = 'I'
  • If yes → send to international hub → stop
  • If no → send for manual inspection → stop
Model full-marks answer (text description):

Start → Read first letter of label → Is it 'D'? Yes → Send to domestic hub → Stop No → Is it 'I'? Yes → Send to international hub → Stop No → Send for manual inspection → Stop
