Programmers use trace tables with algorithms.
Explain the purpose of a trace table.
(2)
(Total for question = 2 marks)

Command word focus: EXPLAIN
  • Say what it shows
  • Explain why it is useful
Mark scheme – ALL acceptable answers:
  • Shows how variable values change step by step (1)
  • Helps identify logic errors (1)
  • Helps understand how an algorithm works (1)
Model full-marks answer:

A trace table shows how variables change as an algorithm runs. It helps programmers understand the algorithm and identify logic errors.

The Airport
An international airport uses a variety of computer systems to deal with passengers, flights and employees.

When an item of luggage leaves the check-in area, it is sorted and delivered to the aircraft.
Luggage is sent to either the international or the domestic sorting hub.

If the label begins with 'I', the item is sent to the international hub. If the label begins with 'D', the item is sent to the domestic hub.
If the label is damaged and not readable, the item is sent for manual inspection.

Draw a flowchart to represent this process for a single item of luggage.
Put a line through any work you do not want to be marked.
(6)

Command word focus: DRAW / REPRESENT
  • Use correct flowchart symbols
  • Include all decision paths
  • Ensure every path ends
Mark scheme – ALL acceptable points:
  • Maximum 3 marks for functionality
  • Maximum 3 marks for accuracy of notation
  • Start symbol included
  • Reads first letter of label
  • Decision: letter = 'D'
  • If yes → send to domestic hub → stop
  • If no → decision: letter = 'I'
  • If yes → send to international hub → stop
  • If no → send for manual inspection → stop
Model full-marks answer (text description):

Start → Read first letter of label → Is it 'D'? Yes → Send to domestic hub → Stop No → Is it 'I'? Yes → Send to international hub → Stop No → Send for manual inspection → Stop
