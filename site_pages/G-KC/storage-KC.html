Data can be stored in different ways.
(i) Describe how data is stored on optical media. (2)
(ii) State the type of secondary storage that stores data as electric charges. (1)
(Total for question = 3 marks)

How to answer (DESCRIBE / STATE):
  • Describe = explain pits/lands and laser
  • State = name storage type
Mark scheme – ALL acceptable answers:
  • Laser burns/heats disk surface (1)
  • Creates pits and lands (1)
  • Pits and lands represent binary data (1)
  • Electric charge storage:
    • Solid state storage (1)
    • Flash storage (1)
    • SSD (allowed)
Model full-marks answer:

Optical media stores data as pits and lands created by a laser on the disc surface. These differences represent binary data. Data stored as electric charges is stored in solid-state (flash) storage.

Computers use different types of storage.
(i) Non-volatile storage keeps its content when the power is turned off. Give one other reason why computers use non-volatile storage. (1)
(ii) State the type of secondary storage media that stores data as a sequence of pits and lands. (1)
(Total for question = 2 marks)

Command word focus: GIVE / STATE
  • One reason for part (i)
  • Name the storage type for part (ii)
Mark scheme – ALL acceptable answers:
  • (i)
    • To store data/files/programs (1)
    • To back up data/files (1)
    • To store BIOS/boot/start-up instructions (1)
    • Higher capacity than volatile storage (1)
    • More portable than volatile storage (1)
  • (ii)
    • Optical storage (1)
    • CD / DVD / Blu-ray Disc allowed
Model full-marks answer:

Non-volatile storage is used to store programs and files. Optical storage stores data as pits and lands.
