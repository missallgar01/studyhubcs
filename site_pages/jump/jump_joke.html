Focus Area Question Answers
Variables What variable is used to keep track of the score? (Where is it created, and how does it change?)
The program uses the variable score. It is created with score = 0 and increases with score = score + 1 whenever the user gets a joke correct.
Input How does the program get answers from the user? (Look for input() and how it’s stored.)
It uses input("> ") to ask the player and stores the result in guess, for example guess = input("> ").strip().lower().
String Methods Why does the program use .strip().lower() after input? (Think about spaces and capital letters.)
.strip().lower() removes extra spaces and turns the answer into lowercase so that things like " Pink Fluff " still match "pink fluff" in the if statement.
If Statement How does the program check if the user is right or wrong? (What line of code compares their answer?)
It uses if statements such as if guess == "pink fluff": and if guess == "a brown stick":. If the guess matches the punchline, it prints a “correct” message and updates the score; otherwise it shows the real answer.
Repetition What parts of the program are repeated for each joke? (How do the jokes follow the same pattern?)
For each joke the program: prints the question, gets input into guess, normalises it with .strip().lower(), checks it with an if/else, and (if correct) adds 1 to score. This pattern repeats for every joke.
Score Update How does the score change when the player gets an answer right? (Find where score increases.)
When the player’s answer is correct, the line score = score + 1 runs to increase the score by 1.
Output (f-string) How does the program show the final score at the end? (Look at the print line with {score}.)
It uses an f-string at the end: print(f"\nYour final score: {score}/3"). The {score} part is replaced with the current value of the score variable.
