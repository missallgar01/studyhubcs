2(b). Question

Binary values stored by a computer can represent different types of data. Discuss the different types of data that can be stored in binary and why computer systems store data in binary format.

You should refer to the following in your answer:

  • what types of data can be represented in binary
  • examples of how binary is used to represent this data
  • why computers store data in binary format.

[9]

Indicative content (mark-scheme points):

Types of data stored in binary (examples):

  • Numeric
  • Alphanumeric / text / characters / symbols
  • Sound
  • Images
  • Video

Examples of how binary represents these:

  • Text: character sets such as ASCII / Extended ASCII (or Unicode). Each character maps to a binary code; more bits → more possible characters.
  • Numbers: stored in binary; negatives can use sign-and-magnitude or two’s complement; real numbers can use floating point.
  • Images: made of pixels; each pixel stores a binary value representing colour.
  • Sound: analogue sound is sampled; each sample is stored as a binary number.

Why computers store data in binary:

  • Computer hardware is built from switches / transistors / logic gates.
  • These have two stable states (on/off), which map naturally to 1/0.
  • Binary is reliable / high tolerance to noise compared to many-level signalling.

Conclusion: Computers store all data in binary because underlying digital hardware works with two states, and all common data types can be encoded into bit patterns.
