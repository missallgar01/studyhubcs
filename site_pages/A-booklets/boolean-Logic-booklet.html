Boolean Expressions – Notes

Boolean Expressions & Notation

Boolean expressions use logical operators to combine variables that can only be 0 (false) or 1 (true).

Operation Symbol Example Meaning
AND ∧ or ∧ A ∧ B True only when both inputs are 1.
OR ∨ or ∨ A ∨ B True when at least one input is 1.
XOR ⊕ A ⊕ B True when exactly one input is 1.
NOT ¬ or overbar ¬A Flips the value of A (0 → 1, 1 → 0).
Equivalence ≡ A ≡ B “A is the same as B” – both have the same truth value.

Difference between OR and XOR:
An OR gate outputs 1 if at least one input is 1; an XOR gate outputs 1 only if exactly one input is 1 (not both).

Truth Tables & Logic Circuits

Every logic gate or Boolean expression can be represented by a truth table listing all possible inputs and the corresponding output.

  • Use column(s) for each input (e.g. A, B, C) and one or more columns for outputs.
  • There are \(2^n\) rows for \(n\) inputs.
  • From a truth table you can derive a Boolean expression, and vice versa.

For questions about circuits, you often need to:

  1. Work out the output of each gate step-by-step.
  2. Fill the truth table.
  3. Write expressions for each output.

Simplifying Boolean Expressions

We can use Boolean algebra rules to make expressions shorter or easier to implement as logic circuits.

General AND rules General OR rules
X · 0 = 0 X + 0 = X
X · 1 = X X + 1 = 1
X · X = X X + X = X
¬X · X = 0 ¬X + X = 1
  • De Morgan’s Laws – break a NOT over brackets and flip AND/OR.
    ¬(A ∧ B) ≡ ¬A ∨ ¬B,   ¬(A ∨ B) ≡ ¬A ∧ ¬B
  • Distribution – expand: A(B + C) ≡ AB + AC
  • Reverse distribution (factoring) – take out common factor: AB + AC ≡ A(B + C)
  • Absorption – e.g. A + A·B ≡ A
  • Double negation – ¬(¬A) ≡ A
  • Commutation / association – reorder and regroup terms.

Example: simplify (¬C ∧ ¬D) ∨ (C ∧ ¬D)

  1. Common factor is ¬D
  2. Factor: ¬D ∧ (¬C ∨ C)
  3. Use X ∨ ¬X = 1 ⇒ ¬D ∧ 1 ≡ ¬D

Karnaugh Maps (K-maps)

A Karnaugh map is a grid representation of a truth table used to simplify Boolean expressions, especially with 3 or 4 variables.

  • Each cell represents one row of the truth table.
  • Cells are arranged so that neighbouring cells differ by only one bit (Gray code).
  • Group 1s in rectangles of size 1, 2, 4, 8… (powers of 2).
  • Each group corresponds to a simplified term; combine all terms with OR.

Typical exam steps:

  1. Highlight the 1s correctly on the K-map.
  2. Form the largest possible groups and write the simplified expression.

D-type Flip-Flops

A flip-flop is a logic circuit that can store one bit of data.

  • Inputs: data input D and a clock input.
  • Output: Q (and often ¬Q).
  • For a D-type flip-flop, Q takes the value of D only on the rising edge of the clock pulse.
  • Between clock edges the output stays the same – this is how a single bit of memory is implemented.

Flip-flops are usually built from NAND gates (AND with an inverted output).

Half Adders & Full Adders

A half adder adds two 1-bit inputs A and B.

  • Outputs: Sum (S) and Carry (C).
  • S = A ⊕ B (XOR), C = A ∧ B (AND).

A full adder also has a carry-in input \(C_{in}\): it adds A, B and \(C_{in}\).

  • Outputs: Sum S and Carry out \(C_{out}\).
  • S = A ⊕ B ⊕ Cin
  • Cout = (A ∧ B) ∨ ((A ∨ B) ∧ Cin)
  • The circuit therefore “adds two bits and a carry bit together” and outputs a 2-bit result (S and Cout).

Check Your Understanding – Questions

1. Writing Boolean expressions

(a) Rewrite the expression NOT(A OR B) XOR C using symbolic notation (¬, ∨, ⊕).
(b) In words, describe what the expression does.

(a) One suitable notation is: ¬(A ∨ B) ⊕ C.

(b) First the OR gate outputs 1 if A or B is 1, then this is inverted. The result is XORed with C, so the final output is 1 if C is different from ¬(A ∨ B).

2. OR vs XOR

Explain the difference in the function of an OR gate and an XOR gate.

An OR gate outputs 1 if at least one of its inputs is 1. An XOR gate outputs 1 if exactly one input is 1 – it is 0 when both inputs are 0 and also 0 when both are 1.

3. Worked simplification

Simplify the Boolean expression (¬C ∧ ¬D) ∨ (C ∧ ¬D) and show your working, naming the rules you use.

  1. Common factor ¬D in both terms:
    (¬C ∧ ¬D) ∨ (C ∧ ¬D) ≡ ¬D ∧ (¬C ∨ C)
  2. Use general OR rule ¬X ∨ X = 1:
    ¬D ∧ (¬C ∨ C) ≡ ¬D ∧ 1
  3. Use general AND rule X ∧ 1 = X:
    ¬D ∧ 1 ≡ ¬D

Final answer: ¬D

4. De Morgan’s law

Simplify the expression so that it does not contain any negation over brackets:
Q ≡ ¬(¬A ∧ ¬B)
Show each step and name the Boolean rule you use.

  1. Apply De Morgan’s law:
    ¬(¬A ∧ ¬B) ≡ ¬(¬A) ∨ ¬(¬B)
  2. Use double negation:
    ¬(¬A) ≡ A and ¬(¬B) ≡ B

So: Q ≡ A ∨ B

5. Factoring an expression

Using Boolean algebra rules, simplify the expression:
(E ∧ F) ∨ (E ∧ G)

Factor out the common term E (reverse distribution):

(E ∧ F) ∨ (E ∧ G) ≡ E ∧ (F ∨ G)

6. Using absorption

Using the rules for manipulating Boolean expressions, simplify:

A ∧ B ∨ A ∧ (B ∨ C) ∨ B ∧ (B ∨ C)

One possible route (summarised):

  1. Distribute where needed and combine like terms so that A∧B terms and B∧C terms are grouped.
  2. Use absorption: (A ∧ B) ∨ B ≡ B and (B ∧ C) ∨ B ≡ B.

Final simplified expression: B ∨ (A ∧ C)

7. Challenge simplification

Simplify the following Boolean expression using De Morgan’s Laws and other Boolean rules. Show enough steps to justify your answer:

¬(A ∧ B) ∨ ¬(A ∧ (B ∨ C)) ∨ (B ∧ (B ∨ C))

After applying De Morgan’s Laws, distribution, absorption and the general AND/OR rules, every possible combination of A, B and C makes the expression true.

Final answer: 1 (the expression is always true)

8. D-type flip-flop

(a) State what a D-type flip-flop does.
(b) When can the output change?
(c) Why is a flip-flop useful in digital systems?

(a) It stores the value of one bit: the output Q copies the input D at certain times and then holds that value.

(b) In a D-type flip-flop, Q can only change on the rising edge of the clock pulse.

(c) Because it holds its value between clock ticks, it can be used as a 1-bit memory cell or register stage inside counters, registers and other sequential circuits.

9. Half adder

(a) Complete the truth table for a half adder with inputs A and B and outputs Sum S and Carry C.
(b) Write Boolean expressions for S and C.

Truth table:

ABSC
0000
0110
1010
1101

Expressions:

  • S = A ⊕ B
  • C = A ∧ B

10. Full adder circuit

A logic circuit has three inputs A, B and Cin and two outputs S and Cout.

(a) Explain what the circuit does, referring to A, B, Cin, S and Cout in your answer.
(b) Write Boolean expressions for S and Cout.

(a) It is a full adder. It adds the three 1-bit inputs A, B and Cin together. The output S is the least significant bit of the sum, and Cout is the carry-out bit produced when the total is 2 or 3.

(b) One common form is:

  • S = A ⊕ B ⊕ Cin
  • Cout = (A ∧ B) ∨ ((A ∨ B) ∧ Cin)
