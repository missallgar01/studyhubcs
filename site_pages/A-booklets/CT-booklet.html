Check Your Understanding – Questions

Question 1(a): A country’s national rail operator wants to represent their rail network on a computer system to keep track of the location of trains, and any problems on the network.

After studying the rail network, the operator uses abstraction to create the virtual representation.

Define the term ‘abstraction’. [1]

Removing characteristics, elements or detail from a problem.

Question 1(b): Identify two reasons why abstraction is needed in the rail network program. [2]

  • Reduce processing requirements
  • Simplify programming
  • Reduce memory requirements

Question 1(c): Describe one potential difference between the virtual and real rail network. [2]

  • Features such as stations or signals may be removed
  • Symbols or keys are used to represent elements
  • The network may not be drawn to scale

Question 2: A group of A-level students are working together to program a computer game.

In the game, the player controls a character who moves through a virtual world. The game starts with a load-up screen. The player can select which area to move to on an on-screen map, and then they control the movements of their character using a keyboard to solve puzzles on the screen.

Explain, using examples, how abstraction would be used to create the virtual world. [4]

  • Unnecessary elements are removed to reduce computational resources
  • The program focuses on the main purpose of the game
  • Characters may be simplified (e.g. stick figures)
  • Scenery such as trees or rivers may be removed

Question 3(a): A car has a feature in its engine management system that is intended to save fuel and emissions produced when the car is waiting at traffic lights or in a traffic jam.

Identify four inputs needed by this feature of the engine management system and suggest a suitable data type for each. [8]

  • Target temperature – Integer / Floating point
  • Wheel movement – Boolean
  • Internal or external temperature – Integer / Floating point
  • Gears engaged – Boolean

Question 3(b): Identify two outputs from this engine management feature. [2]

  • Start engine
  • Stop engine
  • Signal to dashboard display

Question 4: A supermarket uses a stock control system. Details of products are stored on a stock database.

Explain how the system can control the quantity of tins of beans in stock so that the chance of running out is minimised. [6]

  • The barcode is scanned at checkout
  • The stock quantity is reduced
  • The value is compared to a minimum stock level
  • If below minimum and no order exists, a new order is placed
  • An outstanding order is recorded
  • Stock is increased when delivery arrives

Question 5: A flight simulator allows a user to take control of a simulated aeroplane.

Explain what is meant by concurrent processing and describe one example of how the simulator could make use of it. [4]

  • Concurrent processing means processes run at the same time
  • Each plane can move independently
  • Weather systems run simultaneously
  • This is necessary for realism

Question 6(a): A programmer is developing an aeroplane simulator. The user will sit in a cockpit and the simulated environment will be displayed on screens around them.

The programmer uses computational methods to design a solution for the program.

Write a definition for each computational method:

Abstraction
Decomposition [2]

  • Abstraction: Removal of unnecessary components and focusing only on the necessary components
  • Decomposition: Breaking a problem down into smaller sub-problems

Question 6(a)(ii): Give three potential differences between the abstracted aeroplane simulator and reality. [3]

  • Removal of visual elements such as buildings on the ground
  • Simplification of controls
  • Focus on important elements such as weather, height and speed

Question 6(a)(iii): Identify two reasons why abstraction is used when designing a solution to the problem. [2]

  • Reduce memory requirements
  • Reduce processing requirements
  • Simplify the problem being solved

Question 6(b): Describe how caching can be used in the aeroplane simulator. [2]

Data that has been used is stored in cache or RAM in case it is needed again, for example storing weather or cloud designs so they can be accessed more quickly.

Question 7: Functions and procedures are reusable components.

Give two benefits of writing a program with reusable components. [2]

  • Saves time by avoiding rewriting the same algorithms
  • Reduced testing requirements as components can be reused
  • Components can be used in other programs

Question 8: Dexter is leading a programming team who are creating a computer program that will simulate an accident and emergency room to train hospital staff.

Describe what is meant by caching and explain how caching can be used within the simulation. [4]

  • Caching is storing data that has been used in cache or RAM
  • This allows faster access if the data is needed again
  • Patient details or conditions could be stored
  • Room or character designs could be cached

Question 9: Explain two advantages to programmers of using reusable components when developing programs. [4]

  • Components can be reused in future programs
  • They do not need to be rewritten, saving time
  • They have already been tested, improving reliability
  • Programs can be shorter and developed faster

Question 10: A program is needed to plan the layout of a garden.

Explain and evaluate the use of caching and reusable components in the design of the garden program. [9]

  • Caching stores commonly used data in RAM for faster access
  • Garden items or layouts could be cached
  • Reusable components allow shapes or designs to be reused
  • Use of classes enables replication of garden elements
  • Development is faster and performance improves
  • Future adaptations are easier
  • More planning time is required initially

Question 11: A group of A-level students are working together to program a computer game.

Complete the table below, identifying three additional sub-procedures and describing the purpose of each. [6]

  • startGame: Loads the game and displays the load-up screen
  • displayMap: Outputs the map onto the screen
  • selectArea: Allows the user to select an area and loads it

Question 12: A flight simulator allows a user to take control of a simulated aeroplane.

Explain what is meant by concurrent processing and describe one example of how the simulator could make use of it. [4]

  • Concurrent processing is when multiple processes run at the same time
  • Each plane can move independently
  • Weather systems operate simultaneously
  • This is necessary for realism and responsiveness
