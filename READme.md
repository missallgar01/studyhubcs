# studyhubcs

⚠️ Project Status: In active development

This project is under active development. Core functionality is implemented, with ongoing work on additional features, testing, and documentation.

A static revision and learning website for Computer Science content.

## Overview
studyhubcs is a static website designed to organise and present Computer Science revision materials in a clear and accessible format.

It demonstrates clean HTML/CSS structure and content organisation for educational resources.

## Features
- Static HTML pages
- Structured topic navigation
- Clear presentation of technical content
- Automated workflows for site maintenance

## Tech Stack
- HTML
- CSS
- Static site workflows

## Usage
Open `index.html` in a browser or deploy using any static hosting provider.
